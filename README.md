# Sawtooth High Adventure 2026

Living trip guide for the ward Young Men's high adventure — backpacking the
**Alice–Toxaway & Edith Lake Loop** in Idaho's Sawtooth Wilderness,
**August 10–13, 2026**.

**Live site:** https://jedashford.github.io/high-adventure-2026/

## What's here

- `index.html` — the one-page trip guide (itinerary, trail, drive plan, roster, gear, trail intel, open questions)
- `styles.css` — field-guide styling, print-friendly
- `images/` — trail map + elevation profile (AllTrails), driving route (Google Maps), trail photos
